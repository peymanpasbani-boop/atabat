📋 راهنمای تکمیل پنل

۱. تنظیمات سایت — نام سایت، شماره پشتیبانی، توکن تلگرام و درگاه پرداخت را وارد کنید.
۲. شهرهای اعزام — شهرهایی که از آن‌ها اعزام دارید را با نرخ پرواز و اتوبوس تعریف کنید.
۳. پروازها — ایرلاین‌های فعال، ساعات پرواز هر مبدأ و نرخ پایه هر ایرلاین را تعریف کنید.
۴. تاریخ اعزام — تاریخ‌های موجود را اضافه کنید، ظرفیت و افزایش نرخ فصلی را تعیین کنید.
۵. هتل‌ها — برای هر شهر (کربلا، نجف، کاظمین/سامرا) هتل‌های موجود را با درجه و فاصله تا حرم وارد کنید.
۶. نرخ‌گذاری — هزینه پایه هر شب اقامت، ترانسفر، خدمات جانبی و سود هر سفر را مشخص کنید.
۷. کاروان‌های گروهی و خرید گروهی زنده — بسته‌های آماده و کاروان‌های با قیمت پلکانی را معرفی کنید.
۸. تخفیف‌ها و مناسبت‌ها — کدهای تخفیف و رویدادهای ویژه (مثل اربعین) را تعریف کنید.
۹. ذخیره — JSON را دانلود کنید و در مخزن GitHub کنار فایل‌های سایت آپلود کنید.

⚙ تنظیمات کلی سایت

🔑 کلیدهای اتصال (API)

⚠ این مقادیر مستقیماً در فایل JSON ذخیره می‌شوند که در سمت کاربر (مرورگر) بارگذاری می‌شود. برای استفاده واقعی در تولید، توصیه می‌شود درخواست‌های پرداخت و ارسال پیام از طریق یک سرور واسط انجام شود تا کلیدها لو نروند.

✈ شهرهای اعزام و نرخ حمل‌ونقل

نرخ‌ها به تومان و قیمت پایه یک‌طرفه برای هر نفر است. کد فرودگاه برای پنل «پروازها» استفاده می‌شود (مثال: THR). شهرهای «ویژه» در ۴ کارت اول صفحه اصلی نمایش داده می‌شوند؛ بقیه در منوی «سایر شهرها».

✈ ایرلاین‌های فعال

این ایرلاین‌ها در پنل «پرواز نجف» به‌صورت تصادفیِ ثابت (بر اساس تاریخ) بین ۳ تا ۵ پرواز در روز نمایش داده می‌شوند. نرخ پایه به میلیون تومان است.

🕐 زمان‌بندی پرواز هر مبدأ

برای هر شهر مبدأیی که می‌خواهید در پنل پرواز قابل‌انتخاب باشد، ساعات پرواز روزانه و مدت پرواز را وارد کنید. شهرهایی که اینجا تعریف نشوند، در تب‌های پنل پرواز نمایش داده نمی‌شوند.

🏨 هتل‌های سفر انفرادی

برای هر هتل، نام، آدرس، درجه (لوکس / متوسط / اقتصادی) و نرخ هر شب (تومان) را وارد کنید. این نرخ‌ها هنگام دانلود/ذخیره در فایل avan-data.json ذخیره می‌شوند و سایت از همان فایل می‌خواند.

🚌 کاروان‌های گروهی

⚡ کارت‌های خرید گروهی زنده

هرچه نفرات کاروان بیشتر شود، قیمت برای همه پایین‌تر می‌آید. ۴ سطح قیمت را از گران‌ترین (کم‌نفر) تا ارزان‌ترین (پرنفر) وارد کنید — همه به میلیون تومان. آستانه‌های تغییر سطح در «آستانه سطح‌ها» تعیین می‌شود (مثلاً ۹، ۱۹، ۲۹ نفر یعنی سطح ۱ تا ۹ نفر، سطح ۲ از ۱۰ تا ۱۹ نفر و...).

🏷 کدهای تخفیف

کد تخفیف می‌تواند درصدی یا مبلغ ثابت باشد. در صورت خالی‌بودن «تاریخ پایان»، کد تا غیرفعال‌شدن دستی معتبر است.

🎉 مناسبت‌ها و رویدادهای ویژه

مناسبت‌ها (مثل اربعین، نیمه شعبان، محرم) برای برچسب‌گذاری بنر و پیام ویژه روی سایت استفاده می‌شوند. بازه تاریخ به فرمت شمسی YYYY/MM/DD وارد شود.

📦 خروجی JSON برای GitHub

اطلاعات پنل را به‌صورت یک فایل avan-data.json ذخیره کنید و در مخزن GitHub کنار فایل اصلی آپلود کنید. سایت آوان به‌صورت خودکار این فایل را می‌خواند.

📋 راهنمای GitHub

۱. فایل avan-data.json را دانلود کنید.
۲. به مخزن GitHub بروید و فایل را کنار avan_v16.html آپلود کنید.
۳. آدرس raw فایل را در سایت اصلی تنظیم کنید (یا از آدرس پیش‌فرض GitHub Pages استفاده کنید).
۴. برای ویرایش بعدی: همین پنل را باز کنید، فایل JSON قبلی را بارگذاری کنید، تغییر دهید و دوباره دانلود کنید.
